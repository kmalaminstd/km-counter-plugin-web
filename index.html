Documentation

Overview

The KM Counter-Up Plugin is a custom JavaScript feature that animates numerical counters when they come into the viewport. The counter starts from 0 and gradually increases to the specified target number when the section containing the counters is visible. This animation is achieved using the IntersectionObserver API.

Features

  1. Smooth Counter Animation:
    • The numbers gradually count up from 0 to the target value.
  2. Intersection Observer Support:
    • The animation is triggered when the counters come into view, optimizing performance by only animating when visible to the user.
  3. Customizable Animation Duration:
    • You can easily adjust the duration of the counter animation.
  4. Lightweight and fast.
  5. Automatically animates counters when they come into view.

HTML Structure

Each counter is wrapped inside a . km-counter-box container. The number to be animated is contained within an h2 element with the class count-item. The target number is set via the data-target attribute

JavaScript Integration

Make sure to include the minified version of the plugin JavaScript file in your HTML document. Add the script before the closing tag.

https://cdn.jsdelivr.net/gh/kmalaminstd/counterUp_vanilaJs@main/dist/km.counter.min.js
